Introducing Fox Run

Project information

  • Category: Unity 2D Game
  • Client: Niantic Lab
  • Project date: August 2019
  • Language: C#

During the Summer of 2019, I joined the BUILT BY GIRLS program. With this program I was partnered with a mentor who was a Senior Software Engineer at Niantic Inc. I met with this mentor several times and he instructed me to build a game from scratch. After watching some tutorials, doing some readings, and working with my mentor, I was able to create a fully functional game. The game allows for the user to run, jump, crouch, collect coins, die, and ultimately win the game.
